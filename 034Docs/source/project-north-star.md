# Project North Star

本项目的北极星不是数值表、不是重型编辑器，也不是一套万能调试面板。

当前唯一必须被持续放大的核心是：

- 玩家输入的是意图场，而不是直接位移命令
- 集群通过脉冲巡游、节点抓地、回弹与拖拽形成牵引运动
- 圆 / 方 / 三角节点的角色差异必须真实影响运动与捕食
- 收缩 / 舒张必须继续是有效决策
- prey 必须持续逼迫玩家切换姿态，而不是退化成纯 DPS check
- 世界内无文字可读性继续优先于显式说明

本次重构后的开发方向：

- 主流程验证走 `scenario`
- 模块测试走 `playground`
- 正式能力优先落在 `src/modules`
- 运行时协调优先落在 `src/core`
- 旧大 Scene 逻辑只允许通过 `src/legacy/migration-adapters` 过渡

非目标：

- 不做热替换系统
- 不做低配 Unity 式重编辑器
- 不推翻牵引移动核心重写玩法
- 不在结构重构期间继续扩新玩法分支
