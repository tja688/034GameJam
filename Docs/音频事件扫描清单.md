# 音频事件扫描清单（当前实现）

## 1. 扫描范围

本次扫描覆盖以下模块中的“逻辑反馈锚点”与“视觉反馈锚点”：

- `src/sceneInput.js`
- `src/sceneMovement.js`
- `src/sceneCombat.js`
- `src/sceneEnemies.js`
- `src/sceneProgression.js`
- `src/sceneSaveLoad.js`
- `src/sceneTopology.js`
- `src/sceneInit.js`
- `src/ui.js`
- `src/scene.js`

结果已统一收敛到 `src/audioEventRegistry.js`，并通过 `playAudioEvent` 接入 `CoreAudioManager`。

## 4. 后续扩展规则

- 新增音频锚点时，先在 `audioEventRegistry.js` 定义事件，再在业务代码触发。
- 禁止在业务层直接写素材文件名或硬编码播放参数。
- 统一通过事件层 + 管理器层做冷却、并发与总线控制。
